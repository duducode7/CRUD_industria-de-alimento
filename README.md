## Gerenciador de Tarefas - Kanban Simplificado

## Contexto

Uma indústria do ramo alimentício utiliza o Kanban para controlar as tarefas de seus setores nas etapas A Fazer, Fazendo e Pronto.
Com o crescimento das demandas, surgiu a necessidade de uma aplicação web que aumente a visibilidade, integração e controle das tarefas entre os setores.

## Funcionalidades

- Cadastro de usuários 

- Cadastro de tarefas 

- Cadastro de Usuários

- Editar e excluir tarefas

## Tecnologias Utilizadas

- HTML e CSS 

- PHP

- MySQL

## Resumo

O projeto tem como objetivo aplicar os conhecimentos do curso para desenvolver um gerenciador de tarefas simples e funcional, inspirado no modelo Kanban, promovendo organização, transparência e integração entre os setores da empresa.
